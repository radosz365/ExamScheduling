Run

This module provides functionality for generating an exam schedule from a dataset, defining time slots, and creating a visual representation of the schedule for a specific group.

run

dataset = 'dataset1000.csv' module-attribute

The name of the dataset to be used for schedule creation.

file_path = 'schedules/schedule_dataset1000.csv' module-attribute

The path to the generated schedule CSV file.

group_to_display = 'XGD.AR0.ANY.5116.CIL' module-attribute

The identifier of the group to visualize in the schedule.

start_date = '30.01.2025' module-attribute

The starting date for the exam schedule.

time_slots = ['8:00-8:45', '9:00-9:45', '10:00-10:45', '11:00-11:45', '12:00-12:45', '13:00-13:45', '14:00-14:45', '15:00-15:45', '16:00-16:45'] module-attribute

The list of available time slots for exams.
